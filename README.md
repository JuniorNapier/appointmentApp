## Getting Started

To use appointmentApp, please go through below steps.

    Install bower if not already installed.
    
        npm install -g bower

    Clone repository.
    
        git clone https://github.com/JuniorNapier/appointmentApp.git

    Update modules.
    
        npm install
        
        bower install

    Fire up the server.
    
        node app.js

    Access the appointments.
    
        http://localhost:3000
